# Competitor Analysis: UncovAI & Sensity.ai

## Sensity.ai Key Features
- Multilayer detection: pixel analysis + voice signals + GAN detection + file forensics
- **Pixel-Level Manipulation Heatmaps** (visual forensic evidence at pixel level)
- **Frame-by-Frame Face Forgery Labeling** with red bounding boxes + confidence score
- **Segment-Based Visual Explainability** (isolates objects/regions)
- **Precision Landmark Mapping for GAN Exposure** (facial geometry analysis)
- **File Forensic Analysis**: metadata, structural consistency, file generation history
- **File Generation History Tracking**: traces device, software, processing stages
- **Court-ready forensic reports** (admissible in legal proceedings)
- Deepfake Voice Detection: acoustic analysis, formant frequencies, pitch contours
- On-premise deployment options (server, workstation, USB dongle)
- Cloud + on-premise options
- 98% accuracy on public datasets
- 900K+ incidents identified in 2025
- 35 min saved per case
- 30+ countries deployed
- Featured: Financial Times, Forbes, Fortune, Bloomberg, MIT Technology Review

## UncovAI Key Features
- **WhatsApp Bot**: verify suspicious messages, voice notes, images directly from WhatsApp
- **Real-time meeting detection**: Zoom bot + Microsoft Teams bot (auto-joins sessions)
- **Browser Extension**: check authenticity of online articles and social media posts while browsing
- **URL Phishing Protection**: detect AI-written phishing emails, malicious URLs
- **CEO Fraud Detection** use case
- **HR Scam Detection** use case
- Works inside everyday tools (WhatsApp, Zoom, Teams, browser)
- Ecological/sustainable positioning
- API + on-premise deployment
